Setup

Google Drive
Create a Google Cloud project → Enable Drive API → Create OAuth 2.0 Desktop app → Download credentials.json → Import it here.
Open Google Cloud Console
Step-by-step: Google Drive credentials
  1. Open Google Cloud Console.
  2. Create a new project or select an existing project.
  3. Open “APIs & Services” → “Library” → enable “Google Drive API”.
  4. Open “APIs & Services” → “Credentials”.
  5. Click “Create Credentials” → “OAuth client ID”.
  6. Application type: “Desktop app”.
  7. Download the JSON (client secrets).
  8. Back in this app, click “Pick credentials.json” and choose the downloaded file.
  9. Click “Google Drive” above to start sign-in and comparison.
Cleanup and disable access
Use these actions to remove local credentials and revoke/disable access in your Google account and project.
  1. Delete local files: remove credentials.json and the cached token using the buttons above.
  2. Revoke account access: open Google Account → Third‑party access and remove “Escape from Google Drive”.
  3. Disable the API: open Drive API in your project and click “Disable”.
  4. Delete the OAuth client: open APIs & Services → Credentials and delete the Desktop OAuth client you created.
  5. Optional: if this project is only for this app, you can delete the Google Cloud project to remove all credentials and APIs.
